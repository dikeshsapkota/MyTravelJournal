# 🌍 My Travel Journal

A modern and interactive Travel Journal built with **React** and **Vite**. Browse beautiful destinations, explore their details, and even add your own travel experiences through a dynamic form.

---

## 📸 Features

- 🗺️ Explore multiple travel destinations
- ➕ Add your own travel destination
- 🖼️ Upload an image from your device
- 📝 Add title, location, travel dates, and description
- 📍 Store Google Maps links
- 🔁 Data-driven rendering using JavaScript arrays and objects
- ⚛️ Reusable React components
- 🎨 Responsive and modern UI
- ✨ Image hover animations

---

## 🛠️ Built With

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/dikeshsapkota/YOUR_REPOSITORY_NAME.git
```

Navigate to the project

```bash
cd YOUR_REPOSITORY_NAME
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
